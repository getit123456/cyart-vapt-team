# CyArt – Vulnerability Assessment & Penetration Testing (VAPT)

## Repository Name
**cyart-vapt-team**

## Objective
This repository contains hands-on Vulnerability Assessment and Penetration Testing (VAPT) activities conducted as part of CyArt training.

The assessment focuses on identifying, exploiting, and documenting security vulnerabilities in intentionally vulnerable environments such as **Metasploitable2** and **DVWA**.
